Git Out Your Cheat Sheet



Use the following commands in the command line to navigate through files on your local repository.

Navigation

  • cd - Change directories on your hard drive. ex. Possum -> Possum/wordPress
  • cd / - Moves back to root directory.
  • cd .. - Moves up one directory. ex. Possum/wordPress -> Possum

Local Git

  • git init - Create a new repository in a chosen directory.
  • git clone - (local) File path to repository. ex. git clone/AMPPS/wordPress/test
  • git clone - (remote) URL to remote repository. ex git clone https://github.com/username/repository.git

Update That Code

  • git remote add origin "server" - Connect local to remote repository.
  • git pull - Updates local repository with remote changes.

You're Committed

  • git add "file" - Add changes to local files.
  • git add * - Add all changes to local files.
  • git commit -m "Message" - Commit changes.

Check Check 1, 2

  • git status - Check status of current repository.

Don't Forget The Changes

  • git remote add origin "server" - Connect local and remote repositories.
  • git push origin master - Update local changes to remote repository.

The Ten Commitments

  • git log - Display all commits newest to oldest.

There's More Than One Path To The Master

  • git branch "new-branch" - Creates new branch seperate of the master.

Merge Now

  • git merge "branch" - Merge the branch back into a master branch.
